## 📖 Project Overview

An intelligent e-waste management system leveraging artificial intelligence and cloud computing to automate electronic waste classification, tracking, and recycling optimization at Birzeit University, Palestine.

---

## 🎯 Project Objectives

- ✅ Develop AI-based classification system with **93-96% accuracy**
- ✅ Implement real-time device tracking using **IoT sensors**
- ✅ Deploy on **Microsoft Azure** cloud platform
- ✅ Reduce processing time by **70-85%**
- ✅ Decrease landfill contributions by **30-45%**
- ✅ Achieve positive **ROI within 1.6 years**

---

## 🏗️ System Architecture

### Core Components:
- **AI Engine:** CNN-based classification using ResNet-50 transfer learning
- **Cloud Platform:** Microsoft Azure
  - Azure Machine Learning
  - Azure SQL Database
  - Azure IoT Hub
  - Azure App Service
  - Azure Blob Storage
- **Frontend:** React web application
- **Backend:** Flask REST API (Python)
- **Database:** Azure SQL Database
- **IoT:** RFID sensors, weight sensors, GPS trackers

## 🚀 Getting Started

### Prerequisites

- Python 3.9 or higher
- Node.js 16 or higher
- Microsoft Azure subscription
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/AI-EWaste-Management-Birzeit.git

# Navigate to project directory
cd AI-EWaste-Management-Birzeit

# Install Python dependencies
pip install -r requirements.txt

# Install Node.js dependencies (for frontend)
cd src/frontend
npm install
```
